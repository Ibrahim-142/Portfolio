# Muhammad Ibrahim - Portfolio

A modern, responsive portfolio website showcasing my skills, projects, and experience as a Full-Stack Software Engineer. Specializing in MERN stack, Next.js, and AI integrations.

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Installation

```bash
# Clone repository
git clone https://github.com/Ibrahim-142/ibrahim-portfolio.git
cd ibrahimPortfolio

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Features

- Responsive design optimized for all devices
- Smooth scrolling and Framer Motion animations
- Clean and modular component architecture
- Project showcase with technical stacks and direct links
- Modern UI with subtle glassmorphism and glow effects
